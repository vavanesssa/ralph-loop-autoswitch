# UI Core - Spécifications

## Vue d'ensemble
Framework UI principal avec thématisation, layout responsive, composants réutilisables.

## Domaines

### 1. Theme Management
- Dark/light mode toggle
- next-themes provider wrapper
- Persistence localStorage
- Transition de couleurs fluide

### 2. Layout Structure
- Sidebar responsive (collapsible sur mobile)
- Header avec navigation
- Main content area
- Footer

### 3. Core Components
- Button (variations: default, outline, ghost, destructive)
- Input (email, password, search)
- Card (conteneur avec shadow, border)
- Label
- Badge (statuts, notifications)
- ScrollArea (zones de scroll personnalisées)
- Skeleton (loading states)

### 4. Responsive Design
- Mobile-first approach
- Drawer pour sidebar mobile
- Adaptive layouts
- Touch-friendly interactions
